# Cloud Robotics Orchestrator

A streamlined platform for orchestrating ROS/ROS2-based robotic applications in cloud environments. It enables scalable development, simulation, and deployment workflows, bridging the gap between robotics software and modern infrastructure.

## Key Features
*   Manages containerized ROS/ROS2 environments for consistent development and deployment.
*   Provides tools for simulating and testing robotic applications at scale.
*   Orchestrates deployment pipelines for fleets of robots.

## Tech Stack
*   ROS / ROS 2
*   Kubernetes
*   Docker
*   Go / Python

## Getting Started
```bash
git clone https://github.com/zoreanuj/cloud-robotics-orchestrator.git
cd cloud-robotics-orchestrator
make setup
make deploy-local
```
